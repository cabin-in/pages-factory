### `pagesFactory`  

> `pagesFactory` 是基于JSON` 


#### 使用方式

安装：

```bash

 yarn add  pages-factory 
 npm  i  pages-factory --save

```


vue全局引入：

```js

 import pages-factory from 'pages-factory/src/index' //引入源码
 import pages-factory from 'pages-factory' //引入打包文件
 
 Vue.use(pages-factory)

```
